# Jetpack Compose Todo list App

A simple Todo application built with modern Android development tools and best practices.

## About This Project
This project serves as a learning journey into **Native Android Development**. After spending time with **Flutter**, I am now exploring the native ecosystem to understand Jetpack Compose, state management, and the official Android architecture components.

## Tech Stack & Libraries
- **Language**: Kotlin
- **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Dependency Injection**: [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
- **Local Database**: [Room](https://developer.android.com/training/data-storage/room)
- **Navigation**: [Compose Navigation](https://developer.android.com/jetpack/compose/navigation)
- **Architecture**: MVVM / Clean Architecture principles (Data, Domain, Feature layers)

## Features
- Add, update, and manage Todo items.
- Persistent storage using Room database.
- Reactive UI updates using Compose and StateFlow.

## Learning Goals
- Transitioning from Flutter's declarative UI to Jetpack Compose.
- Mastering Kotlin Coroutines and Flow.
- Implementing Dependency Injection with Hilt.
- Understanding the official Android Architecture Guidelines.
